    Lesson 1 — Before writing code¶

    This page exists for one reason:

    to make sure you feel ready
    before you write your first Python instruction.

    No pressure.
    No rush.

    What you are about to do¶

    In Lesson 1, you will: - write your first Python code - see immediate results - understand what the computer is doing

    You are not expected to: - understand everything - remember everything - be perfect

    You are expected to: - try - run the code - observe the output

    That’s it.

    Programming is not magic¶

    Programming is often presented as something complex.

    In reality, it is very simple:

    you write instructions
    the computer follows them

    The computer does not: - think - guess - understand intentions

    It only follows exact instructions.

    This is good news.

    How Python helps beginners¶

    Python was designed to: - be readable - be simple - avoid unnecessary symbols

    When you read Python code, you should be able to say:

    “I can guess what this does.”

    That is not an accident.
    That is the design.

    How to approach this lesson¶

    Here is the right mindset:

    • Don’t rush
    • Don’t skip running the code
    • Don’t copy answers immediately
    • Look at the output carefully

    Learning Python is not about speed.
    It’s about understanding cause and effect.

    A note for Network Engineers¶

    If you work in networking, you already think logically.

    You already: - follow procedures - apply configurations step by step - verify outputs

    Programming is the same mindset.

    The only difference: - instead of devices - you give instructions to the computer

    That’s why Python fits networking so well.

    What matters most in the beginning¶

    At the start, only two things matter:

    1. Writing code
    2. Reading output

    Everything else comes later.

    Sparky note

    Clean output is how humans trust automation.
    We start with output for a reason.

    You are ready¶

    If this page makes sense to you, you are ready.

    ➡️ Go to Lesson 1 — Print & Output
    and write your first Python instruction.
